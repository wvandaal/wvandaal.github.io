About Me

Allow me to (re)introduce myself...

My name is Willem and I'm a web developer experienced in HTML5, CSS3, JavaScript, Ruby on Rails, and responsive web design.

I recently graduated from Trinity College with degrees in Computer Science and Hispanic Studies. I first started designing for the web in the winter of 2013 and I haven't looked back. Of particular interest to me is the rapidly evolving landscape of web design, especially in the areas of Shadow DOM, Web Components, and Responsive Design. In addition to Rails and JavaScript, I have experience in jQuery, Sass, SQL, and Backbone.js, and test-driven development in RSpec.

PwrSlide

HTML5 Presentation Web Components

PwrSlide was designed to allow programmers to easily create powerful presentations by using HTML, CSS, and JavaScript. It was created using the new Web Components spec, which allows the use of custom HTML tags, each with their own unique functionality.

Asteroids

Pure JS Arcade Fun

A redesign of the classic arcade game, my implementation of Asteroids uses the HTML5 canvas element and pure JavaScript to handle the in-game physics and object rendering. Asteroid frequency and HP scale relative to the player score to provide realistic skill curve.
