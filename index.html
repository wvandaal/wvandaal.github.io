Jot.

Single-Page Markdown Editor

Jot. is an open-source note-taking application and markdown editor built with Backbone.js and Rails 4. This application features syntax highlighting, live markdown rendering, PDF downloads, and live preview following, automatically positioning the output window to the location of the most recent user change. PwrSlide

HTML5 Presentation Web Components

PwrSlide is an open-source, responsive framework designed to allow programmers to easily create powerful presentations by using HTML, CSS, and JavaScript. It was created using the new web components spec and the Polymer.js library, which allows the use of custom HTML tags, each with their own unique style and functionality.

Muse

A Smarter Music Aggregator

A music discovery web application using Mechanize, Nokogiri and Feedzirra for HTML and RSS/Atom feed parsing to discover newly posted songs on music blogs. Muse features Facebook login with Devise and Omniauth, AJAX endless pagination, Jaccardian-similarity based recommendations implemented with Sidekiq and Redis.

Asteroids

Pure JS Arcade Fun

A redesign of the classic arcade game, my implementation of Asteroids uses the HTML5 canvas element and pure JavaScript (with the revealing module pattern) to handle the in-game physics and object rendering. Asteroid frequency and HP scale relative to the player score to provide a realistic skill curve for the user.

Rails Lite

Lightweight Clone of Rails 3.2

Rails-Lite was built as a learning exercise to gain a greater understanding of the inner workings of Ruby on Rails. It implements RESTful routing with parameter parsing, session handling, CSRF authenticity tokens, the Rails ControllerBase object with redirection support, ERB template rendering, and automatic URL helper generation.

ActiveRecord Lite

Lightweight Clone of ActiveRecord

Similar to Rails-Lite, I built this project as a learning exercise to gain a greater understanding of the inner workings of ActiveRecord. It implements Model objects, attribute assignment and mass-assignment protection, associations (has_many, belongs_to, and has_one_through), and basic ActiveRecord queries (find, where, all, and save).
